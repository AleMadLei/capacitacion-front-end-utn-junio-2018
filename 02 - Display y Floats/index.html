display

Valores por defecto

Las etiquetas tienen un valor de display por defecto.

  • block (div, p, section)
  • inline (a, span, img)
  • none (definido por el desarrollador)

Valores por defecto

Existen otros tipos de display, cada uno con propiedades que ayudan a hacer cosas que antes eran más complicadas.

  • flex
  • grid
  • list-item (li)
  • table, table-row, table-cell

Float

Propósito

Para dar estructura visual de contenido (layout).

Hacer que los elementos "floten" (salen del flujo normal del contenido).

Uso común, colocar imagenes a la par del texto y que se vea como en revistas.

Ejemplo sin float

Este es un ejemplo de texto que demuestra que es lo que pasa cuando la imagen no tiene un float. Como pueden observar, la imagen puede alinearse al texto (depende de otras propiedades de CSS) y queda un espacio visual incómodo.

Ejemplo con float left

Este es un ejemplo de texto que demuestra que es lo que pasa cuando la imagen tiene un float a la izquierda. Como pueden observar, la imagen da ese efecto de artículo de periódico o revista.

Este es un ejemplo de texto que demuestra que es lo que pasa cuando la imagen tiene un float a la izquierda. Como pueden observar, la imagen da ese efecto de artículo de periódico o revista.
