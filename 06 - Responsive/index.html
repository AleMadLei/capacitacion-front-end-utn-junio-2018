Responsive

Qué es?

Diseño que adapta su contenido dependiendo de las características del dispositivo con el cual se este visualizando.

Cómo se hace?

  • En un mundo perfecto, es parte del diseño original.
  • En un mundo perfecto, mobile first.
  • Se escribe el HTML, se define el CSS base.
  • Media Queries
  • Son parte del CSS

Código

Traducción muy descriptiva

@media Condiciones de Activación {   SelectorDeCSS {
    propiedad: valor;
  }
}

Ejemplos de Media Query

@media print {
  #alemadlei-code-1 {
    text-transform: uppercase;
  }
}

Ejemplos de Media Query

@media screen and (min-width: 700px) {
  #alemadlei-code-2 {
    color: red;
  }
}

Ejemplos de Media Query

@media screen and (max-width: 960px) {
  #alemadlei-code-3 {
    background: yellow;
  }
}

Ejemplos de Media Query

@media screen and (min-width: 780px) and (max-width: 810px) and (min-height: 400px) and (max-height: 450px) {
  #alemadlei-code-4 {
    background: yellow;
  }
}

Notas

  • No deberían ser muy complicados.
  • Lo ideal es trabajar de menos a más (de mobile a desktop).
  • En resoluciones menores, lo usual es "apilar" elementos.
  • Considerar casos particulares (iPhone y iPad con los hover).
