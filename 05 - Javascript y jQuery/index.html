Notas generales

  • NO ES JAVA!!!
  • HTML + CSS != programar, pero JS == programar.
  • Lenguage de programación
  • Funcionalidad basada en eventos
  • Actualmente se guía bajo la especificación ECMAScript
  • Lenguage para hablar de cosas relacionadas al lenguaje (dom, closure, promise, loop)

Recomendaciones

  • Es bueno tener bases fuertes en el lenguaje para usar otras tecnologías (Angular, React, jQuery, NodeJS, etc).
  • Tomar en cuenta que puede haber discrepancias entre navegadores.
  • Incluir los scripts al final del HTML (especialmente si se realizan modificaciones a estilos).
