BEM

Metodología

Block Element Modifier

  • Ordernar el CSS para que a largo plazo sea más mantenible.
  • Reducir los errores a la hora de trabajar en equipos.
  • Reducir los errores a la hora de agregar elementos nuevos.
  • Facilitar la lectura del CSS e interpretar la posible estructura del HTML.
  • Utilizar selectores de clase.
  • Más información en el sitio de BEM.

Block

  • El bloque.
  • La base del nombre de clase a utilizar.
  • A grandes rasgos es sinónimo de componente.
  • Reutilizable.
  • Componentes pueden ser parte de otros componentes.
  • .bloque { ... }

Block

Element

  • El elemento.
  • Siempre pertenece a un bloque y solo a ese bloque.
  • Se identifica por el doble unserscore (__) entre el nombre del bloque y el nombre del elemento.
  • .bloque__elemento { ... }

Element

Modifier

  • El modificador.
  • Altera una visualización base de un bloque o un elemento.
  • Pueden haber varios modificadores.
  • Se identifica por el doble guion (--) que se agrega al final del nombre del bloque o el elemento.
  • .bloque--modificador,
    .bloque__elemento--modificador { ... }

Modifier

Position

Funcionalidad

Indica la forma en que se van a posicionar los elementos a la hora de desplegarse en el documento.

  • static
  • relative
  • absolute
  • fixed
  • sticky

static

  • Tipo de posicionamiento por defecto.
  • Colocado en el flujo normal de la página
  • No es afectado por otras propiedades de posicionamiento (left, right, top, bottom, z-index).
Esta es mi posición por defecto (static).

Esta es mi posición por defecto (static).

relative

  • Es relativo a su posición por defecto (static).
  • Se coloca en base a las otras propiedades de posicionamiento.
Tengo left 20px.
Tengo right 20px.
Tengo bottom 20px.

absolute

  • Es absolute respecto al primer ancestro que este posicionado (que no sea static).
  • En caso de no existir un ancestro posicionado, será con respecto al body.
Tengo left -20px y top -20px.
Tengo right -20px.
Tengo bottom 20px.

fixed

  • Es fijo con respecto a la pantalla.
  • No se ve afectado por el scroll.
Tengo left -20px y top -20px.
Tengo right -20px.
Tengo bottom 20px.

sticky

  • Es fijo con respecto al ancestro posicionado.
  • Es relativamente nueva y no sirve en algunas versiones o requiere prefijos de proveedor.
Tengo left -20px y top 5px.
Tengo top 30px right 20px.
Tengo right 10px y bottom 20px.
